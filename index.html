Local

Operaciones relacionadas con los locales de tu empresa.

Listar locales

Devuelve una lista de locales filtrados por nombre, estado y ordenados por prioridad o nombre.

Authorizations:
api_key
query Parameters
term
string
Example: term=Salón

Busca por el nombre del local. El nombre puede ser parcial o total.

isActive
boolean
Example: isActive=true

Indica si se deben listar solo los locales activos. true para activos, false para inactivos. Si no se envía, se traen todos los locales.

sortKey
string
Example: sortKey=priority

Campo por el cual se ordenará el listado. Puede ser name o priority. Si no se envía, por defecto es priority.

productType
required
string (ProductTypeEnum)
Enum Description
SIMPLE

Cuando un producto no tiene un padre. Por ejemplo: Coca cola

CONFIGURABLE

Cuando es un producto padre que asocia variantes. Este producto no se vende porque es solo un agrupador. Por ejemplo: Camisa. La cual puede tener variantes como Color y tamaño

CONFIGURABLE_HIJO

Cuando es una variante el producto configurable. Este producto si se vende. Por ejemplo: Camisa Roja Small

COMBO

Cuando es un producto agrupador de varios productos con secciones a vender. Por ejemplo: Hamburguesa con papas

SERVICIO

Cuando es un producto no físico. Por ejemplo: Mantenimiento de horno
